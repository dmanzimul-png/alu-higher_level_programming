# JavaScript Warm Up

## Description
This project introduces the fundamentals of JavaScript programming using Node.js. It covers basic scripting concepts such as variables, data types, functions, loops, conditionals, arrays, objects, modules, and command-line arguments.

The project is part of the **ALU Higher Level Programming** curriculum and aims to build a strong foundation in JavaScript before moving on to more advanced concepts and web development.

---

## Learning Objectives

At the end of this project, I should be able to explain:

- Why JavaScript programming is amazing
- How to run a JavaScript script using Node.js
- How to create variables and constants
- The differences between `var`, `let`, and `const`
- The available JavaScript data types
- How to use `if`, `if...else`, and loops
- How to write and use functions
- What scope means in JavaScript
- How to use arithmetic operators
- How to manipulate objects and arrays
- How to import and export files using modules

---

## Requirements

- Ubuntu 20.04 LTS
- Node.js v14.x
- Allowed editors:
  - vi
  - vim
  - emacs
  - Visual Studio Code
- All files end with a new line.
- The first line of all JavaScript files is:

```javascript
#!/usr/bin/node
```

- Code follows **Semistandard Style (Standard + Semicolons)**.
- All files are executable.

---

## Project Structure

| File | Description |
|------|-------------|
| `0-javascript_is_amazing.js` | Prints "JavaScript is amazing" |
| `1-multi_languages.js` | Prints three programming language messages |
| `2-arguments.js` | Prints a message based on the number of arguments |
| `3-value_argument.js` | Prints the first argument |
| `4-concat.js` | Prints two arguments in a sentence |
| `5-to_integer.js` | Converts an argument to an integer |
| `6-multi_languages_loop.js` | Prints messages using an array and a loop |
| `7-multi_c.js` | Prints "C is fun" multiple times |
| `8-square.js` | Prints a square using the character `X` |
| `9-add.js` | Adds two integers |
| `10-factorial.js` | Computes a factorial recursively |
| `11-second_biggest.js` | Finds the second biggest integer |
| `12-object.js` | Updates an object property |
| `13-add.js` | Exports an addition function |

---

## Usage

Run a script:

```bash
node filename.js
```

Example:

```bash
node 0-javascript_is_amazing.js
```
