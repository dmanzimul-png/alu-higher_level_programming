# JavaScript - Objects, Scopes and Closures

## Description
This project introduces JavaScript object-oriented programming concepts, including:

- Classes and objects
- Constructors
- Instance methods
- Inheritance using `extends`
- Closures
- Modules and `require`
- Exporting functions and classes
- Manipulating arrays and objects

The project is part of the **ALU Higher Level Programming** curriculum and focuses on understanding how JavaScript handles objects, scopes, and closures.

---

## Learning Objectives
At the end of this project, I should be able to explain:

- Why JavaScript programming is amazing
- How to create an object in JavaScript
- What `this` means
- What `undefined` means
- Why variable types and scope are important
- What a closure is
- What a prototype is
- How inheritance works in JavaScript
- How to use classes and object-oriented programming concepts

## Requirements

- Ubuntu 20.04 LTS
- Node.js v14.x
- Allowed editors:
  - vi
  - vim
  - emacs
  - Visual Studio Code
- All files end with a new line.
- The first line of all files is:

```javascript
#!/usr/bin/node
```

- Code follows Standard JavaScript style (semistandard).

## Project Structure

| File | Description |
|------|-------------|
| `0-rectangle.js` | Empty Rectangle class |
| `1-rectangle.js` | Rectangle with width and height |
| `2-rectangle.js` | Rectangle with input validation |
| `3-rectangle.js` | Rectangle with print method |
| `4-rectangle.js` | Rectangle with rotate and double methods |
| `5-square.js` | Square class inheriting from Rectangle |
| `6-square.js` | Square class with charPrint method |
| `7-occurrences.js` | Counts occurrences in a list |
| `8-esrever.js` | Reverses a list without using reverse() |
| `9-logme.js` | Logs the number of arguments printed |
| `10-converter.js` | Converts numbers to different bases |
