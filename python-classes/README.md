# Python Classes Project (OOP Basics)

## Description

This project introduces Object-Oriented Programming (OOP) in Python by building a `Square` class step by step. Each task adds new functionality such as attributes, validation, methods, and printing.

## Learning Objectives

* Understand what a class and object are
* Learn how to use attributes and methods
* Understand private attributes and why they are used
* Implement validation using setters
* Use properties (`@property`) in Python
* Apply basic OOP concepts like encapsulation

## Files

* `0-square.py` → Empty class definition
* `1-square.py` → Square with private size attribute
* `2-square.py` → Size validation (type and value)
* `3-square.py` → Area calculation
* `4-square.py` → Getter and setter for size
* `5-square.py` → Print square using `#`
* `6-square.py` → Square with position support

## How to Run

Make sure Python 3 is installed.

Run a test file like this:

```bash
python3 0-main.py
```

Replace `0-main.py` with the corresponding test file for each task.

## Notes

* No external modules were used
* All attributes are validated where required
* Code follows basic Python style guidelines
* Each class and method includes documentation
