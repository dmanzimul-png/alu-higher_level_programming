# Python Everything is Object - ALU Higher Level Programming

## Project Overview

This project explores the concept that **everything in Python is an object**. You will learn about:

- Python object types (`type()`)
- Object identifiers (`id()`)
- Mutable vs. immutable objects
- Equality (`==`) vs identity (`is`)
- How objects behave when passed to functions
- Memory management and object references

This project consists of multiple small exercises where you answer questions, predict outputs, or write small Python functions.
