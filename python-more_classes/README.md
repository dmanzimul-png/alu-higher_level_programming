# Python More Classes - Rectangle Project

## Description

This project builds on Object-Oriented Programming (OOP) concepts in Python by implementing a `Rectangle` class step by step. Each task introduces new features such as validation, methods, class attributes, and special methods.

## Learning Objectives

* Understand class and instance attributes
* Use getters and setters for validation
* Implement methods like `area` and `perimeter`
* Work with special methods like `__str__`, `__repr__`, and `__del__`
* Use class attributes and track instances
* Understand static methods and class methods

## Files

* `0-rectangle.py` → Empty Rectangle class
* `1-rectangle.py` → Rectangle with width and height validation
* `2-rectangle.py` → Area and perimeter methods
* `3-rectangle.py` → String representation using `#`
* `4-rectangle.py` → Object recreation using `__repr__`
* `5-rectangle.py` → Detect instance deletion
* `6-rectangle.py` → Track number of instances
* `7-rectangle.py` → Custom print symbol
* `8-rectangle.py` → Compare rectangles using static method
* `9-rectangle.py` → Create square using class method

## How to Run

Make sure Python 3 is installed.

Run a test file like this:

```bash
python3 0-main.py
```

Replace `0-main.py` with the corresponding test file for each task.

## Features

* Input validation for width and height
* Area and perimeter calculation
* Custom string representation
* Instance tracking
* Rectangle comparison
* Square creation from rectangle
